Login Process

Login process involves below steps :
  1. Admin Login
  2. User Login

  • Admin login is basically administrator login which can be only done by administrator of this cyber cafe.
  • User login is basically a normal or local user login which can also be termed as guest login.
  • Login is done by user so that their browsing session can be started and then they will be charged according to the time spent by them
    which will be calculated once the user logs out.
  • In case the user, forgots his/her password then they can get password through forgot password link or can ask admin to provide the same.

Also if user is new, then they can register themselves first and then login and start browsing the internet.
